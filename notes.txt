  Table in HTML
      In HTML we can display multiple data in the format of table(combination of rows and 
      coloums) by using the table tag itself.
      1) We can create a table with border by using TABLE tag and BORDER attribute.
      2) Inside the table we can create a row by using TABLE ROW (tr) tag.
      3) In frist row of table we insert a headers using TABLE HEADER (th) tag.
      4) In rest of the rows of a table we insert a data using TABLE DATA (td) tag.

      <TABLE>
           <TR>  <TH>  </TR>
           <TR>  <TD>  </TR>
           <TR>  <TD>  </TR>
      <TABLE>
